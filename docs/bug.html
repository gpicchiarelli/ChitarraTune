Segnala un bugReport a bug

Grazie per l’aiuto a migliorare ChitarraTune. Seguono alcuni consigli per aprire una segnalazione efficace.

Thanks for helping improve ChitarraTune. Here are a few tips to file an effective report.

Prima di aprireBefore you file

  • Aggiorna all’ultima release e riprova.
  • Cerca issue esistenti simili.
  • Annota i passaggi per riprodurre il problema e l’hardware usato (microfono/interfaccia).
  • Update to the latest release and try again.
  • Search for similar existing issues.
  • Write down exact reproduction steps and your audio hardware (mic/interface).

Informazioni utili da includereUseful information to include

  • Versione app (dal pannello “Informazioni” o dal sito) e commit/tag, macOS (es. 14.5), Mac modello.
  • Preset accordatura, calibrazione A4, modalità (Auto/Manuale), dispositivo audio selezionato.
  • Passi per riprodurre, risultato atteso vs ottenuto, eventuali screenshot.
  • App version (About panel or website) and commit/tag, macOS (e.g. 14.5), Mac model.
  • Tuning preset, A4 calibration, mode (Auto/Manual), selected audio device.
  • Steps to reproduce, expected vs actual result, screenshots if any.

PrivacyPrivacy

Non includere informazioni personali o dati sensibili. L’app elabora l’audio in locale; allega solo ciò che desideri condividere pubblicamente.

Do not include personal or sensitive data. The app processes audio locally; only attach content you’re comfortable sharing publicly.
